1.概要
	・DirextXのエンジンを使用してスーパーマリオギャラクシーを参考に制作した3Dアクションゲームです。
      ・Player自身が主人公となりお姫様を助けるために様々なステージを冒険します。
      ・主人公が光を放ちながら宇宙を探索します。

2.開発環境

　　　・ハードウェア: WindowsPC(i7,メモリ32GB)
　　　・開発用ＯＳ: Windows10
　　　・統合開発環境： Microsoft Visual Studio 2019
　　　・開発言語： C / C++ / HLSL
      ・ライブラリ: directX11 / FBXSDK / ImGui


3.作品に対するアピールポイント
　　　
　　　・エフェクト表示はビルボードを使用した板ポリを用いています。
　　　・オブジェクトの設置,画像の配置、カメラ遷移の仕方などをImGuiを用いてプランナーやデザイナーでも気軽に変更・追加することができます。
　　　・各ステージの情報を外部に保存することでコードの可読性などがあがりました。
　　　・オブジェクト指向を用いることで敵の作成・ブロックの作成・ステージの作成など安易に出来るように工夫しました。
　　　・カメラワークや細かなオブジェクトに対して線形補間を使うことで気持ちのいい操作性・見た目にしました。
　　　・Worldシーンでのギミック成功時のカメラワークを工夫しました。
　　　・敵に攻撃を与える際にヒットストップ演出をいれることで,ヒットしたことが認識しやすくなるようにしました。
　　　・Playerの状態にステートパターンを用いることでコードの可読性・拡張性を高くしました。
　　　・シェーダで画面遷移・看板を作成しました。
      ・関数内で値が変わらない引数は const をつけ,値の変更がないことをわかりやすくしました。
      ・定数の定義の仕方をcppの上にnamespaceを作りstatic constで作成するように統一しています。
      ・UIの移動などに自作で開発したイージングコンポーネントを使うことで、緩急がつけやすいようになっています。
　　　　　　
4.反省点、改善点

　    ・四角のオブジェクトに斜め上から当たってしまうとめり込んでしまう。
　　　　　　　ー＞自身の軸でレイを回転させて移動方向(右、左、前、下)にレイを飛ばしているので
　　　　　　　　   斜めから当たってしまうのでめり込んでしまう。
　　　

      ・デザイナーやプランナーが調整しやすいように工夫したい。
		　->ImGuiでの調整・作成方法を研究する。
